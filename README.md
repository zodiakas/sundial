### Python Script for Shepherd's or Cylindrical Sundial Generation
#### Overview
This is a Python script designed for the generation of shepherd's or cylindrical sundial, historically referred to as pillar dial, cylinder, or chilindre. These devices are portable altitude dials with a rich history that extends from Antiquity through the Renaissance.
#### Script Functionality
The script accepts user input for key parameters: latitude, time zone, cylinder diameter (in millimetres), and cylinder height (in millimetres). Based on these inputs, it calculates the required gnomon and produces an editable dial in an Inkscape .svg file format.
#### Historical Significance
The shepherd's or cylindrical sundial has held considerable importance throughout history, symbolising both practical timekeeping and intellectual sophistication. During the Renaissance, these sundials were celebrated for their artistic and scientific merit, frequently appearing in prominent works of art and literature
#### Principle of Operation
The pillar dial functions by measuring the sun's altitude rather than its azimuth. This allows the user to find the time without a magnetic compass, provided the current date is known.
1.	Date Alignment: The user rotates the top of the dial (the gnomon support) until the gnomon is positioned directly over the vertical line corresponding to the current date or Zodiacal sign.
2.	Gnomon Extension: The gnomon is extended horizontally, perpendicular to the cylinder.
3.	Vertical Suspension: The dial is suspended by its top knob or string so that it hangs perfectly vertical.
4.	Solar Alignment: The user rotates the entire suspended assembly until the gnomon is pointed directly toward the sun. This is achieved when the gnomon casts a narrow, vertical shadow straight down the side of the cylinder.
5.	Reading the Time: The time is indicated by the position of the tip of the shadow. The point where the shadow's end falls among the arcuate hour lines reveals the local apparent solar time.
